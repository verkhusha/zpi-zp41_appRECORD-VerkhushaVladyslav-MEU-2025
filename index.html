Скрін-шот таблиці



У цій таблиці використовуються наступні теги та атрибути:

Теги

  • <table> - визначає таблицю в HTML.
  • <thead> - контейнер для заголовків стовпців таблиці, який містить <tr> із заголовками (<th>).
  • <tbody> - контейнер для основного вмісту таблиці, який містить рядки таблиці (<tr>), що складаються з комірок (<td>).
  • <tr> - представляє рядок у таблиці.
  • <th> - визначає заголовок стовпця в таблиці.
  • <td> - визначає комірку в таблиці для основного вмісту.

Атрибути

<class> - використовується для стилізації елементів.

HTML-код зображення та скрін-шот цього зображення.



Скрін-шот зображень.



Теги.

<img> - елемент для вставки зображення. Використовується для відображення фото ресторану в галереї.

Атрибути.

src (у <img>) - вказує шлях до зображення, яке буде відображено на сторінці. Наприклад, src="./img/photo1.jpg" вказує шлях до першого зображення. alt (у <img>) - альтернативний текст для зображення, який відображається, якщо зображення недоступне або не завантажилося. Тут він описує кожне фото ресторану, наприклад, alt="Фото ресторану 1".
